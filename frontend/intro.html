Problem

우리가 해결하는 문제

배송 기사들이 매일 겪는 비효율과 고충

01
비효율적인 수동 경로 계획

수십 개의 배송지를 수작업으로 정리하느라 업무 시작 전부터 소모되는 시간과 에너지

02
과도한 연속 운전

적절한 휴식 없이 이어지는 장시간 운전으로 인한 피로 누적과 사고 위험

03
배송 현황 파악의 어려움

관리자가 실시간 배송 현황을 파악하기 어려워 발생하는 소통 비용과 오차

Features

핵심 기능

루트온이 제공하는 스마트 솔루션

🗺️
다중 경유지 경로 최적화

카카오맵 API와 OR-Tools로 최대 10개 경유지의 최적 순서를 자동 계산해 이동 시간과 연료비를 절감합니다.

☕
스마트 휴식 포인트 추천

배송 횟수와 주행 시간 기준으로 경로 이탈을 최소화하는 최적의 휴식 장소를 자동으로 추천합니다.

📍
GPS 기반 자동 완료

앱이 GPS를 주기적으로 전송하면 서버가 50m 반경 도착을 감지해 배송을 자동 완료 처리합니다.

📊
관리자 실시간 모니터링

관리자 웹에서 전체 배송 기사의 위치와 배송 현황을 실시간으로 확인하고 배송지를 배정합니다.

🔐
역할 기반 접근 제어

관리자와 배송 기사 계정을 분리하고 JWT 인증으로 각 역할에 맞는 기능만 안전하게 접근합니다.

⚡
고성능 데이터 파이프라인

TimescaleDB로 GPS 시계열 데이터를 효율적으로 저장하고, Redis로 실시간 위치를 빠르게 처리합니다.
