FEATURES

화물 운송에 필요한
모든 것

🗺️
경로 최적화
OR-Tools TSP 알고리즘으로 다중 경유지 방문 순서를 자동으로 최적화합니다. 카카오 모빌리티 API 기반 실제 도로 소요 시간을 반영합니다.
핵심
😴
법정 휴게 자동 삽입
화물자동차 운수사업법 [별표3] 기준으로 1시간 40분 누적 운전 시 휴게소를 자동으로 경로에 삽입합니다.
📡
실시간 관제
WebSocket 기반 실시간 GPS 수신으로 기사의 현재 위치를 지도에 즉시 표시합니다. 50m 도착 감지로 배송 완료를 자동 처리합니다.
🚚
원격 배차
운행 중 관리자가 새 경유지를 추가하면 기사 앱에 즉시 알림을 전송하고 자동으로 경로를 재계산합니다.
⚡
재경로 계산
정체, 사고 등 돌발 상황 발생 시 현재 위치 기준으로 즉시 재최적화합니다. 긴급 예외 모드로 3시간까지 연속 운전을 허용합니다.
📊
경로선 시각화
카카오맵 위에 실제 도로 경로선을 표시합니다. 출발지, 경유지, 휴게소, 목적지를 구분된 마커로 확인할 수 있습니다.
HOW IT WORKS

간단한 4단계

01
📋
운행 등록
관리자가 경유지와 목적지를 등록하고 기사에게 배차합니다.
→
02
🧠
경로 최적화
기사가 출발 시 OR-Tools TSP로 최적 방문 순서와 휴게소 위치를 계산합니다.
→
03
🗺️
운행 시작
기사는 최적 경로로 운행하고 5초마다 GPS를 전송합니다. 관리자는 실시간으로 위치를 확인합니다.
→
04
✅
자동 완료
배송지 50m 이내 도착 시 자동으로 완료 처리됩니다. 관리자는 원격으로 경유지를 추가할 수 있습니다.
